# DPapprox

Dynamic Programming-inspired algorithm for approximation of discrete inputs based on the relaxed solution.

## Building

```
git clone https://github.com/raminabbasi/DPapprox.git
cd DPapprox
mkdir && cd build
cmake ..
make
make install
```
